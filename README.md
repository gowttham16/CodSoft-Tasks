# To-Do List Application (Python)

##  Project Description
This is a simple Command-Line Interface (CLI) based To-Do List application developed using Python.  
The application allows users to add, view, and remove tasks efficiently.

This project was created as part of my Python Programming Internship Task.

---

##  Features
- Add new tasks
- View existing tasks
- Remove tasks
- User-friendly menu-driven interface
- Continuous execution until user exits

---

##  Technologies Used
- Python 3
- VS Code

---

##  How to Run the Project

1. Install Python on your system.
2. Open the project folder in VS Code.
3. Open terminal inside the folder.
4. Run the command:

   python todo_list.py

5. Follow the menu instructions.

---

##  Sample Output

The program displays a menu like:

1. Add Task  
2. View Tasks  
3. Remove Task  
4. Exit  

Users can select options and manage tasks easily through the terminal.
